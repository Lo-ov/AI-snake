## WebGL 贪吃蛇（可试玩）

### 运行方式

- **方式 A（推荐，最简单）**：双击打开 `snake/index.html`。
- **方式 B（静态服务器）**：如果你的浏览器对本地资源有额外限制，可在项目根目录执行：

```bash
python -m http.server 5173
```

然后访问 `http://localhost:5173/snake/`。

### 试玩操作手册

- **开始 / 暂停**：`Space`
- **重开**：`R`
- **移动方向**：
  - 方向键：↑ ↓ ← →
  - 或 `W A S D`

### 规则说明

- 吃到红色食物会 **加分并增长**。
- 每次吃到食物会 **略微加速**。
- **撞墙** 或 **撞到自己** 即游戏结束。

### 界面说明

- 顶部展示：`Score`（当前分）、`Best`（最高分，本地存储）、`Speed`（相对速度）。
